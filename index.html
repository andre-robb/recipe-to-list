Recipe to List

Paste ingredients → copy to Apple Reminders

    Copies your list and runs the Grocery to Reminders shortcut automatically. First time? Set up the shortcut below.

    Create a shortcut that reads your clipboard and adds each line as a separate reminder.

    1. Open the Shortcuts app on your Mac.
    2. Click + to create a new shortcut. Name it Grocery to Reminders.
    3. Add the action Get Clipboard.
    4. Add the action Split Text — set separator to New Lines.
    5. Add the action Repeat with Each. Inside the loop, add Add New Reminder — set the title to Repeat Item and choose your grocery list.
    6. Save. Next time: copy your list here, then run the shortcut from the menu bar or with a keyboard shortcut.
